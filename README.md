## Red Cross Linux Setup
For my [Eagle Scout Project](https://stem.rathhacks.tech), I receved 30+ Laptops from the America Red Cross, and to give them away at my event to the kids/attendee's, I installed the following on the laptops to make them function on there low 1–3GB DDR3 Ram and i-3—i-5 processers from 2012.
- [OS: Lubuntu](https://lubuntu.me/) - Lubuntu is an official, lightweight, and fast Ubuntu Linux distribution that uses the LXQt desktop environment instead of the heavier GNOME interface. (Open Source)
- [Browser: Min](https://minbrowser.org/) - Min is an open-source, lightweight web browser built with Electron (CSS and JavaScript) designed for speed, privacy, and minimalism. (Open Source)
- [Disk Cloning: Clonezilla](https://clonezilla.org/) - Clonezilla is a free, open-source software suite designed for disk imaging, partition cloning, and bare-metal backup/recovery. It enables users to copy entire disks or specific partitions, supporting various operating systems (Windows, Linux, macOS) and file systems. (Open Source)
